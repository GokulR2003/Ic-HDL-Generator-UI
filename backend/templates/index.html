Quick Start Guide

1

Choose Your Tool

Start with the Circuit Designer for visual design, IC Database to explore components, or Boolean Logic Tool for expression-based design.

2

Design Your Circuit

In the designer, drag components from the sidebar, place them on canvas, and use the wire tool to connect pins. Switch between Select and Wire modes using the toolbar.

3

Generate & Export

Click "Export HDL" to generate Verilog or VHDL code. Use "Save" to store your design for later. All generated code is production-ready!
